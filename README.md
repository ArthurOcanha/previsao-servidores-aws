# 📈 Previsão de Capacidade de Servidores AWS

Projeto acadêmico de análise preditiva para antecipar a necessidade de escalonamento de infraestrutura em nuvem (AWS), simulando um cenário real de aplicativo de corridas em crescimento.

Desenvolvido durante o curso de **Tecnologia em Inteligência Artificial — FIAP (2026)**.

---

## 🎯 Objetivo

Antecipar datas críticas em que a demanda de um aplicativo ultrapassaria a capacidade atual dos servidores, permitindo planejar o escalonamento antes de ocorrerem falhas.

## 🛠️ Tecnologias utilizadas

| Ferramenta | Uso |
|---|---|
| Python | Linguagem principal |
| Pandas | Manipulação e limpeza dos dados |
| NumPy | Operações numéricas |
| Matplotlib | Visualização de séries temporais |
| Pathlib | Gerenciamento de caminhos de arquivo |

## 📋 Etapas do projeto

1. **Análise exploratória** — entendimento e limpeza do dataset fornecido em aula
2. **Modelagem de tendência** — identificação do padrão de crescimento ao longo do tempo
3. **Projeção de datas críticas** — cálculo dos momentos em que a capacidade seria atingida
4. **Visualização** — gráficos de séries temporais com formatação profissional de datas
